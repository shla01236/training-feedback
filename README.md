> ## 培训反馈收集工具
>
> 专为培训现场设计的实时互动工具。培训师发起一个话题，学员扫描二维码打开填写页面提交关键词或感受，内容以词云形式实时展示在大屏幕白板上，频次越高字号越大。
>
> **主要功能**
> - 每次收集生成独立话题和二维码，投屏即可使用
> - 词云实时更新，相同内容自动合并计数
> - 学员可对他人已提交内容点 +1 表示认同
> - 支持多次收集，历史记录可查看和删除
> - 管理端需登录，学员扫码无需注册
>
> **技术栈**：Node.js · Express · WebSocket · WordCloud2.js
